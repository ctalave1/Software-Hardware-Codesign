Equality Operators

Operators that compare the equality of two operands

  1. Equal (==)

    Is true if both operands being compared are equal in value.

    Ex: 1 == 1

  2. Not Equal (!=)

    Is true if both operands are not equal in value.

    Ex: 1 != 2

Relation Operators

Operators that compare the values of two operands

  1. Less Than (<)

    Is true when the left operand's value is less than the right operand's

    Ex: 4 < 9

  2. Greater Than (>)

    Is true when the left operand's value is greater than the right operand's

    Ex: 5 > 1

  3. Greater Than Or Equal To (>=)

    Is true when the left operand's value is greater than or equal to the right operand's

    Ex: 5 >= 5

  4. Less Than Or Equal To (<=)

    Is true when the left operand's value is less than or equal the right operand's

    Ex: 3 <= 5

Bitwise Operators

Operators that compare the bits of two operands

  1. Bitwise AND (&)

    Compares the bits of two operands and copies a bit to the result if it exists in both of the operands

    Ex: 0001 & 1100 = 0000

  2. Bitwise OR (|)

    Compares the bits of two operands and copies a bit to the result if it exists in the other operans

    Ex: 0001 | 1100 = 1101

  3. Bitwise NOT (~)

    Flips the bits of an operand

    Ex: ~1001 = 0110

  4. Bitwise XOR (^)

    Compares the bits of two operands and copies the bit to the result only if the bit is set (1) in one of the operands, but not both

    Ex: 1011 ^ 1110 = 0101

Logical Operators

Operators that compare two boolean operands

  1. Logical AND (&&)

    Compares the boolean value of two operands and is true only when both values are true

    Ex: T && F = F

  2. Logical OR (||)

    Compares the boolean value of two operands and is true only when both or one of the operands are true

    Ex: T || F =T

  3. Logical NOT (!)

    Flips the boolean value of an operand

    Ex: !T = F
